🔐 Cybersecurity Operations Dashboard

This project demonstrates a complete penetration testing workflow conducted in a controlled virtual lab using Kali Linux and Metasploitable.

📊 Project Overview

The goal of this project was to perform network reconnaissance, identify vulnerabilities, exploit them, and visualize the results using Power BI.

🧠 Penetration Testing Workflow

  • Network scanning using Netcat and Nmap
  • Service enumeration and analysis
  • Identification of vulnerable services
  • Exploitation using Metasploit Framework
  • Successful root-level system access achieved

🚨 Key Findings

  • Multiple insecure services detected (FTP, Telnet, SMTP, HTTP)
  • Critical vulnerability: UnrealIRCd backdoor (CVE-2010-2075)
  • Weak authentication via Telnet
  • Successful exploitation with root shell access

🛠 Technologies Used

  • Kali Linux
  • Nmap & Netcat
  • Metasploit Framework
  • Power BI
  • VMware
